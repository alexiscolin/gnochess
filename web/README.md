# gnochess Front-end

MVP front-end made with [HUGO](https://gohugo.io/) / [Vite](https://vitejs.dev/) / and Vanilla Typescript/Javascript + [sevejs](https://github.com/alexiscolin/seve-js) for components system management.

## Install

```bash
npm install
```

## Start Server

```bash
npm run dev
```

## Build

```bash
npm run build
```
